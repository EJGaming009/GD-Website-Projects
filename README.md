How to import a level into geometrydash.com

Step 1: If your on Windows/Linux, Click Ctrl+Shift+I, On MacOS click Cmd+Option+I

You can also do right click and click inspect

Step 2: Open the Network Tab, Refresh the page, and find the file "1.txt"

Step 3: Right Click "1.txt" and click Override Content.

You may need to select a folder for it to save to.

Step 4: Make sure the text in the file for the level you want to import is copyed

Step 5: Go back to geometrydash.com and delete everything in the "1.txt" file

Then, you can past the new text in and type in "allow pasting" (if you need to)

Step 6: Now you should be able to paste it in. Then Press Ctrl+S (Cmd+S on MacOS)

Then your done! Refresh the page then when its done loaded refresh.

How to import your own level

You should probably read the how to import if you havent yet.

Step 1: You need geode on Geometry dash, then you need the GDShare mod.

Step 2: Now get the level you want to import. Keep in mind that you are only limited to a few objects to work with.

(This includes Cube and Ship portal, Slab, Block, Other Block cuz idk what to call it lol, Spike, Flat Spike, Small Spike, etc)

Step 3: Export the level.

Step 4: Open the File.

Now for this part, if you now about XML then you are good.

If not, then all you need to know is to copy the string from the "<s>" up to the "</s>"

For example:

<?xml version="1.0"><dict><s>LONG LEVEL STRING YOU COPY</s></dict>

Then just copy the long level string and paste it into the "1.txt"

Then Press Ctrl+S (Cmd+S on MacOS)

Then your done! Refresh the page then when its done loaded refresh.
